--PÁRRAFOS--

Un párrafo es un fragmento escrito que tiene una unidad de sentido

En HTML se utiliza la etiqueta p para especificar donde empieza y termina un parrafo.
Para realizar un ralizar un salto de línea se utiliza la etiqueta "br"


--LINKS/ENLACES--

Los Link y/o Enlaces se crean con la etiqueta "a" con los atributos:
"href"donde se coloca la ubicacion o el URL de la pagina destino.
"target" donde se especifica con si se abrirá una nueva pantalla o se abrirá en la misma.
Se puede agregar un texto para el Link, el cual se va a desplegar en la pagina.

--IMÁGENES--

Se pueden insertar Imágenes con la etiqueta "img" y los atributos:
"src" donde se ingresa la fuente de la imagen
"alt" Para personas con debilidad visual
"widht" para el ancho de la imagen en pixeles
"Height" para la altura de la imagen en pixeles


--IMÁGENES CON LINK--

Las imágenes se pueden utilizar como enlace hacia un Link, agregando una etiqueta de imagen (img) dentro de una etiqueta de link (a)


TABLAS

Las tablas se crean con la etiqueta "table":
Para agregar filas a la tabla se utiliza "tr".
Para agregar los titulos se utiliza "th".
Para agregar la informacion en las filas se utiliza "td".

--LISTAS--

Para crear una lista sin orden especifico, solo con viñeta se utiliza la etiqueta "ul".
Para agregar un elemento a la lista se utiliza la etiqueta "li".

Para crear una lista con orden jerarquico, enumerado se utiliza la etiqueta "ol".
Para agregar un elemento a la lista se utiliza la etiqueta "li".

Para creae una lista con descripcion de cada elemento se utiliza la etiqueta "dl".
Para agregar un titulo o elemento se utiliza "dt"
Para agregar una descripcion al elemento se utiliza "dd"
